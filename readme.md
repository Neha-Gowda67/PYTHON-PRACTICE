File Organizer (Python)

This project automatically organizes files in a folder into categories like Images, Documents, Videos, and Others.
 How to Run
1. Run the Python file
2. Enter folder path
3. Files will be sorted automatically
 Output Example

Enter the folder path: test_folder

abc.jpg moved to Images  
doc.pdf moved to Documents  
video.mp4 moved to Videos  
notes.txt moved to Documents  

Files organized successfully!

 Result
Files are organized into:
- Images
- Documents
- Videos
- Others
